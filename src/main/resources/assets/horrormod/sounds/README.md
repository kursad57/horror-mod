Place your .ogg sound files for the mod here.

- night_amb.ogg -> referenced by `assets/horrormod/sounds.json` as `horrormod:night_amb` (ambient loop)
- jumpscare.ogg -> referenced by `assets/horrormod/sounds.json` as `horrormod:jumpscare` (short loud effect)

- whisper.ogg -> referenced by `assets/horrormod/sounds.json` as `horrormod:whisper` (soft ambient whisper used for message triggers)
Notes:
- Do NOT include copyrighted music in the mod distribution unless you have explicit rights. If you want ambient music, add your own royalty-free file and reference it in `sounds.json`.

After adding .ogg files, rebuild the mod with the Forge MDK/Gradle setup so the assets are packaged.
